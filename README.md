This is the Repository for the DB25 control software made for the Arduino.

To make use of the Code, the .axaml file is the front end, and the .axaml.cs is the back end. 

This is running off of the Avalonia.NET package to make use of Linux and windows compatibility. 

Do the following:

1. Install Avalonia UI.NET package
2. Create a project using Avalonia UI in Visual studio
3. Copy the contents of the .axaml file and the .axaml.cs file into the generated project files.
